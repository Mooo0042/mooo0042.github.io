Über mich

Hallo! Ich bin ein Hobby-Entwickler, der gerne an verschiedenen Projekten arbeitet. Mit etwas Erfahrung in mehreren Programmiersprachen und mithilfe von LLMs bringe ich Ideen zum Leben.

Beim Programmieren nutze ich LLMs, um mich zu unterstützen. Natürlich prüfe ich den Code, damit nirgendwo ein Access Token oder Passwort steht.

Meine Schwerpunkte liegen in der Software- und Minecraft-Mod-Entwicklung, aber ich experimentiere auch gerne mit anderen Sprachen und Frameworks. Hier findest du eine Auswahl meiner Projekte.

Meine Projekte

Be A DJ (Frontend)

Ein Projekt für den Winterball an unserer Schule, das es den Teilnehmern ermöglicht, ihre Musik in die Playlist einzufügen. Mit einem Moderations- und Blocksystem, um sicherzustellen, dass nur die richtigen Inhalte hinzugefügt werden.

React PostgreSQL

Be A DJ (Backend)

Das Backend für den Winterball an unserer Schule. Verwaltet die Datenbank, API-Endpoints und die gesamte Server-Logik für das Musikwunsch-System mit PostgreSQL-Anbindung.

Node.js Express PostgreSQL

ShotsByUs3r0

Eine Instagram-ähnliche App, die alle Bildverhältnisse darstellen kann und Downloads erlaubt, die auch deaktiviert werden können. Darüber hinaus gibt es eine Kommentar-, Like- und Share-Funktion.

React PostgreSQL

Iam Music Player Renewed

Eine Minecraft-Mod, die eine Boombox zum Spiel hinzufügt. Die Boombox ermöglicht es Spielern, Musik von YouTube, SoundCloud und HTTP-URLs zu spielen und zu hören. Diese Mod wurde von einem anderen Entwickler entwickelt und wurde von mir nur übernommen.

Java

CSV Tool

Tool, welches CSV-Dateien automatisch so umschreibt, dass diese in IServ importiert werden können.

C++

Edura

Webbrowser basierend auf Chromium Embedded Framework (CEF) für Kiosk-Geräte, der es nicht erlaubt, Hyperlinks und Buttons zu benutzen. Dazu wurde die Markier- und Zoom-Funktion entfernt.

C++ C CEF
